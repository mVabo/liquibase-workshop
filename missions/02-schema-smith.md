# Mission 02 - Schema Smith

## Outcome
You will ship a small schema optimization using correct Liquibase workflow:
1. Read and validate existing constraints.
2. Add a new index migration.
3. Apply and verify it in PostgreSQL.

## Prerequisites
- Mission 01 is complete.
- Your app starts cleanly with Liquibase.

## Step 1 - Review existing schema migrations
Open:
- `src/main/resources/db/changelog/missions/001-create-core-tables.yaml`
- `src/main/resources/db/changelog/missions/003-add-potions-description.yaml`

Identify:
- Primary keys on all core tables.
- Foreign keys from `orders` and `order_items`.
- Unique constraints on `customers.email`, `potions.code`, and `order_items(order_id, potion_id)`.

## Step 2 - Confirm constraints in the live database
In psql, run:

```sql
SELECT tc.table_name, tc.constraint_name, tc.constraint_type
FROM information_schema.table_constraints tc
WHERE tc.table_schema = 'public'
  AND tc.table_name IN ('customers', 'potions', 'orders', 'order_items')
ORDER BY tc.table_name, tc.constraint_type, tc.constraint_name;
```

## Step 3 - Create an index migration
Create file:
`src/main/resources/db/changelog/missions/004-add-orders-created-at-index.yaml`

```yaml
databaseChangeLog:
  - changeSet:
      id: 011-add-orders-created-at-index
      author: workshop
      changes:
        - createIndex:
            tableName: orders
            indexName: idx_orders_created_at
            columns:
              - column:
                  name: created_at
      rollback:
        - dropIndex:
            tableName: orders
            indexName: idx_orders_created_at
```

## Step 4 - Register the new migration
Add this include to `src/main/resources/db/changelog/db.changelog-master.yaml` after mission `003`:

```yaml
  - include:
      file: missions/004-add-orders-created-at-index.yaml
      relativeToChangelogFile: true
```

## Step 5 - Apply migration
Restart the app:

```bash
mvn spring-boot:run
```

## Step 6 - Verify index exists
In psql:

```sql
SELECT indexname, indexdef
FROM pg_indexes
WHERE schemaname = 'public'
  AND tablename = 'orders'
ORDER BY indexname;
```

Expected: `idx_orders_created_at` is listed.

## Done Criteria
- New changelog file is added and included in master.
- Changeset `011-add-orders-created-at-index` from `004-add-orders-created-at-index.yaml` is executed.
- Liquibase runs without error.
- `idx_orders_created_at` exists in PostgreSQL.
